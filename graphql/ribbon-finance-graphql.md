# Ribbon Finance GraphQL API

Ribbon Finance provides a GraphQL API via The Graph protocol, giving developers access to indexed on-chain data for Ribbon Finance Theta Vaults on Ethereum mainnet. The subgraph tracks vault performance metrics, fee collection, premium earnings, per-share pricing history, option trades, depositor accounts, and balance updates — enabling historical APY analysis and vault activity queries without running a full archive node.

**Endpoint:** https://api.thegraph.com/subgraphs/name/ribbon-finance/ribbon-v2

**Documentation:** https://docs.ribbon.finance/developers/ribbon-subgraph
